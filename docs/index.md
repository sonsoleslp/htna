Heterogeneous Transition Network Analysis (HTNA) for sequences that mix
two or more actor groups (e.g. Human and AI) within the same
interaction. Provides a thin, focused API on top of the 'Nestimate'
estimation engine and the 'cograph' rendering engine: build a transition
network over the combined sequence while preserving the actor partition,
so downstream bootstrap, permutation, reliability, centrality, and
plotting functions treat each actor's codes as a distinct node group.
